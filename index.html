Adjusted img width

To avoid changing the proportion of an image, we can focus instead on just changing the width of the image. This is done by setting the height:auto, width to whatever size you want it to be. It's best to add a width and height to images even if you don't plan on changing them. The reasoning is that by adding a width and height, the page will load the white space it plans to take up, and won't cause sudden slow down once the rest of the content loads in.

box-sizing: border-box

The purpose of border-box involves the box model, where the padding and border are included in the height of the element. With border-box, if you enter width: 100px; height:100px;the element will be exactly 100px by 100px. To showcase this, if you have padding:10px; border:10px solid; height:100px; width:100px .Without border box the total width and height will be 140px x 140px since we gave the content 100px x 100px but then the padding and border are added after, changing the height and width. With border box, the height x width of the content will be 60px x 60px; with the padding and border taking up their 20px x 20px each.The height and width of the element of the whole will now be 100px x 100px as intended

To further explore this, this is taken from Web Docs. "In the standard box model, if you give a box a width and a height attribute, this defines the width and height of the content box. Any padding and border is then added to that width and height to get the total size taken up by the box. The actual space taken up by the box will be 410px wide (350 + 25 + 25 + 5 + 5) and 210px high (150 + 25 + 25 + 5 + 5)."

.box { width: 350px; height: 150px; margin: 10px; padding: 25px; border: 5px solid black;}

Outer display

Display Block

An outer display. A block will take up its own line, it will take up the full width of the container, and border,margin, and padding will push things away from it.

Display inline

An outer display.A new line won't be made, height and width won't apply, only horizontal padding,margins, and border will push things away.

Inner Display

Display: Flex

With Display: Flex the outer display will be block,but the inner display will now be flexed.

Margin Collapsing

If two elements margin touch, they won't combine, they would just be the marin of the larger of the two. If one is negative, then they would subtract from the larger of the two. "The top paragraph has a margin-bottom of 50 pixels. The second paragraph has a margin-top of 30 pixels. The margins have collapsed together so the actual margin between the boxes is 50 pixels and not the total of the two margins." - Web Doc
